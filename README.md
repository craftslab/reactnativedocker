## Introduction

*React Native Docker* is the Dockerfile for Android & React Native SDK built on Ubuntu.



## Build

```bash
./build.sh
```



## Deploy

```bash
docker pull craftslab/reactnativedocker:latest
```



## Run

```bash
./run.sh
```
